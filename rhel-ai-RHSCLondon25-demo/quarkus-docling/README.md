# Quarkus docling-java extension example

Create your quarkus docling-java project

```bash
mvn io.quarkus.platform:quarkus-maven-plugin:3.31.4:create \
  -DprojectGroupId=org.appdev.cop.pamenon \
  -DprojectArtifactId=docling-app \
  -Dextensions="rest,io.quarkiverse.docling:quarkus-docling:1.2.2"
```
